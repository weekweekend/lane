## react-vite-template

这是一个基于 Vite 的 React 项目。

### 技术栈

- Vite: https://cn.vitejs.dev/
- React 18: https://reactjs.org/
- ReactRouter: https://reactrouter.com/

### 开发

```bash
## 安装依赖
npm install

## 启动项目
npm run dev

## 构建正式环境
npm run build

## 预览构建产物
npm run preview
```
